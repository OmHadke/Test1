Core User Roles & Flows

Three clean roles keep development simple and easy to explain.

Registration Flow

  1. Create restaurant profile with hours + contact.
  2. Upload dish images, ingredients, calories, price.
  3. Generate QR code linked to live menu.

Operations Flow

  1. Confirm table bookings in real time.
  2. Accept pre-orders & mark availability.
  3. Review feedback and star ratings.

QR Menu Flow

  1. Scan restaurant QR code.
  2. Browse dishes with ingredients + calories.
  3. Add items to cart & place order.

Booking Flow

  1. Select date, time, guests.
  2. Receive confirmation instantly.
  3. Leave a rating after dining.

Governance

  • Verify new restaurants.
  • Monitor compliance & data quality.
  • Run dashboards for platform insights.

Support

  • Manage complaints and dispute resolution.
  • Track system uptime and SLA.
  • Maintain payment gateway configs.
